# Take Home Test for Web Devs

## Time Tracker

- The **objective** is to allow a team to track their time as quickly as possible (few clicks, efficient UI)
- Users should be able to sign up and log in.
- It should allow adding a time entry, with a number of hours and task description and project. Projects can be a static, pre-filled list. The current date and time should automatically be recorded.
- The user wants to be able to see their work for the week and totals for each project.
